# UI Timeline Notes

- There is one primary global ruler.
- Only the selected lane may show a local timing overlay.
- The local timing overlay must remain visually subordinate to the global ruler.
- Free-running lanes must be visibly distinct from lanes aligned to the global ruler.

Milestone 6 implements three visible module lanes and one selected-lane local timing overlay. The overlay remains visually subordinate to the global ruler, and only the selected lane receives it.
